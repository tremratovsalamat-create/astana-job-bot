# 🤖 Astana Jobs Bot

Telegram-бот для поиска актуальных вакансий с hh.kz прямо внутри Telegram.

---

## ✨ Возможности

| Функция | Описание |
|---|---|
| 🔍 **Поиск вакансий** | Динамический поиск по любой профессии на hh.kz |
| 🏙 **Популярные профессии** | Быстрый выбор из топ-профессий Астаны |
| 👍 **Лайк / Дизлайк** | Сохраняй понравившиеся вакансии или пролистывай |
| ⭐ **Сохранённые вакансии** | Список всех лайкнутых вакансий |
| 📄 **Загрузка резюме** | Хранение PDF-резюме в боте |
| 🚫 **Фильтр дублей** | Просмотренные вакансии не показываются повторно |
| 💾 **SQLite база данных** | Все данные хранятся локально |

---

## 🚀 Быстрый старт

### 1. Клонируй репозиторий
```bash
git clone https://github.com/yourusername/astana-jobs-bot.git
cd astana-jobs-bot
```

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Настрой токен
```bash
cp .env.example .env
```
Открой `.env` и вставь токен от [@BotFather](https://t.me/BotFather):
```
BOT_TOKEN=123456789:AAF...your_token_here
```

### 4. Запусти бота
```bash
python bot.py
```

---

## 🛠 Как получить BOT_TOKEN

1. Открой Telegram и найди **@BotFather**
2. Напиши `/newbot`
3. Задай имя: `Astana Jobs Bot`
4. Задай username: `astana_jobs_bot` (или любой свободный)
5. Скопируй токен и вставь в `.env`

## 📁 Структура проекта

```
astana_jobs_bot/
├── bot.py          # Главный файл — хендлеры Telegram
├── parser.py       # Парсер вакансий с hh.kz
├── database.py     # SQLite — все операции с БД
├── config.py       # Настройки (токен, лимиты)
├── requirements.txt
├── Procfile        # Для деплоя на Render/Railway
├── .env.example    # Шаблон переменных окружения
└── .gitignore
```

---

## 🌐 Деплой на Render (бесплатно)

1. Создай аккаунт на [render.com](https://render.com)
2. Нажми **New → Web Service**
3. Подключи свой GitHub репозиторий
4. Настройки:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `python bot.py`
5. В разделе **Environment** добавь:
   - `BOT_TOKEN` = твой токен
6. Нажми **Deploy**

### Деплой на Railway

1. Зайди на [railway.app](https://railway.app)
2. **New Project → Deploy from GitHub**
3. Добавь переменную `BOT_TOKEN`
4. Railway сам подхватит `Procfile`

## 🗄 База данных

SQLite база `jobs_bot.db` создаётся автоматически.

| Таблица | Описание |
|---|---|
| `users` | Пользователи бота |
| `saved_jobs` | Сохранённые (лайкнутые) вакансии |
| `seen_jobs` | Просмотренные вакансии (дедупликация) |
| `resumes` | Загруженные резюме (file_id Telegram) |
| `search_history` | История поисковых запросов |

- **Python 3.11+**
